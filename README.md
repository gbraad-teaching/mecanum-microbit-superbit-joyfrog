Micro:bit (Mecanum) Omni-bot - Using Yahboom Super:bit + KittenBot Joyfrog protocol
===================================================================================

Requires
--------
  * Yahboom Super:bit
  * Yahboom Omni:bot
  * 2x Micro:bit
  * KittenBot Joyfrog

Usage
-----

  * Upload the standard JoyFrog code to the controller
    `controller.ts`
  * Upload the modified superbit code to the car
    `superbit.ts`
